# ZuneWP

A modern, fluid Android music player built with Jetpack Compose, designed to bring back the nostalgia, clean typography, and minimalist aesthetic of the classic Microsoft Zune interface and Windows Phone "Metro" UI.

## Features

- **Zune/Metro Inspired UI:** Large, bold typography, panoramic smooth side-scrolling navigation, and minimal, uncluttered layouts.
- **Jetpack Compose Native:** Built 100% using declarative UI for optimal performance and buttery-smooth animations.
- **Dynamic Backgrounds:** Immersive layouts featuring smooth shifts and artist-focused art treatments matching the original media player vibe.
- **Lightweight Media Core:** Engineered to index and play back local audio storage efficiently.

## Tech Stack

- **Language:** Kotlin
- **UI Framework:** Jetpack Compose (Material 3 / Custom UI Design System)
- **Architecture:** MVVM (Model-View-ViewModel) + Clean Architecture
- **Dependency Injection:** Hilt (Optional/Planned)
- **Asynchronous Flow:** Kotlin Coroutines & StateFlow
